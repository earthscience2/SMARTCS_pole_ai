## interpolate_data_to_grid() 상세 프로세스

```
┌─────────────────────────────────────────────────────────────────┐
│          interpolate_data_to_grid() 시작                         │
│  입력: df, meas_range, target_heights, target_degrees           │
└─────────────────────────────────────────────────────────────────┘
                            │
                            ▼
        ┌───────────────────────────────────────┐
        │  1. 원본 데이터 높이 계산              │
        │     heights = linspace(stheight,       │
        │                        edheight,       │
        │                        total_rows)     │
        └───────────────────────────────────────┘
                            │
                            ▼
        ┌───────────────────────────────────────┐
        │  2. 채널별 각도 계산                   │
        │     - 측정 범위(stdegree~eddegree)     │
        │     - 8개 채널을 균등 분배             │
        │     - 예: 0~90도 → ch1=10도,          │
        │              ch2=20도, ..., ch8=80도   │
        └───────────────────────────────────────┘
                            │
                            ▼
        ┌───────────────────────────────────────┐
        │  3. 각 목표 높이별로 반복              │
        │     (target_heights)                  │
        └───────────────────────────────────────┘
                            │
                            ▼
        ┌───────────────────────────────────────┐
        │  4. 높이 범위 내 확인                  │
        │     if target_height < min_h or       │
        │        target_height > max_h:          │
        │        continue                        │
        └───────────────────────────────────────┘
                            │
                            ▼
        ┌───────────────────────────────────────┐
        │  5. 각 채널별 높이 보간                │
        │     - interp1d로 높이 방향 보간        │
        │     - channel_value_at_height 저장     │
        └───────────────────────────────────────┘
                            │
                            ▼
        ┌───────────────────────────────────────┐
        │  6. 각 목표 각도별로 반복              │
        │     (target_degrees)                  │
        └───────────────────────────────────────┘
                            │
                            ▼
        ┌───────────────────────────────────────┐
        │  7. 각도 방향 보간                     │
        │     - 채널별 각도와 값으로             │
        │     - interp1d로 각도 방향 보간        │
        │     - 360도 경계 처리                  │
        └───────────────────────────────────────┘
                            │
                            ▼
        ┌───────────────────────────────────────┐
        │  8. 보간 실패 시 처리                  │
        │     - 가장 가까운 채널 값 사용         │
        └───────────────────────────────────────┘
                            │
                            ▼
        ┌───────────────────────────────────────┐
        │  9. 결과 저장                          │
        │     row_data = {                      │
        │       'height': target_height,         │
        │       'degree': target_degree,        │
        │       'value': interpolated_value     │
        │     }                                  │
        └───────────────────────────────────────┘

                            ▼
        ┌───────────────────────────────────────┐
        │  10. DataFrame 생성 및 반환            │
        │      (height, degree, value)          │
        └───────────────────────────────────────┘
```

## 주요 데이터 구조

### 입력 데이터
- **CSV 파일**: `{poleid}_{index}_{date}_OUT_{axis}_breakheight_{height}_breakdegree_{degree}.csv`
  - 컬럼: `idx`, `groupname`, `poleid`, `devicetype`, `measno`, `ch1`, `ch2`, ..., `ch8`, `axis`

### 중간 데이터
- **all_measurements**: `{measno: {'x': DataFrame, 'y': DataFrame, 'z': DataFrame, 'meas_range': dict}}`
- **break_info.json**: `{'poleid': str, 'project_name': str, 'breakstate': str, 'breakheight': float, 'breakdegree': float, 'measurements': {...}}`

### 출력 데이터
- **최종 CSV**: `{poleid}_OUT_merged.csv`
  - 컬럼: `height`, `degree`, `x_value`, `y_value`, `z_value`, `devicetype`

## 주요 검증 조건

1. **각도 범위 검증**: `abs(angle_range) >= 90.0` (90도 이상만 머지)
2. **높이 범위 검증**: `target_height`가 `min(stheight, edheight)` ~ `max(stheight, edheight)` 범위 내
3. **데이터 완전성**: x, y, z 데이터가 모두 있어야 머지 가능
4. **채널 데이터**: ch1~ch8이 모두 있어야 보간 가능

## 보간 방법

1. **높이 방향 보간**: `scipy.interpolate.interp1d` (linear)
2. **각도 방향 보간**: `scipy.interpolate.interp1d` (linear, 360도 경계 처리)
3. **보간 실패 시**: 가장 가까운 채널의 값 사용
